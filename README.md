# xrwvm-fullstack_developer_capstone

## Project Name
Full-stack Developer Capstone - Car Dealership Application

## Repository Name
xrwvm-fullstack_developer_capstone

## Description
A full-stack web application for car dealerships built with Django, React, Node.js, and MongoDB. Features include dealer listings, review management, sentiment analysis, and user authentication.

## Technologies
- **Backend**: Django, Flask, Node.js/Express
- **Frontend**: React, Bootstrap
- **Database**: SQLite, MongoDB
- **Cloud**: IBM Cloud Code Engine
- **CI/CD**: GitHub Actions

## Features
- View car dealerships across the US
- Filter dealers by state
- User registration and authentication
- Submit and view dealer reviews
- Sentiment analysis on reviews
- Admin panel for content management

## Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
